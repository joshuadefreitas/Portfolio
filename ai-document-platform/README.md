# AI Document Intelligence & Financial Workflow Platform

Transforming messy financial documents into clean data, insights, and intelligent automation.

## Phase 1 — Portfolio MVP

- Upload PDF/CSV invoices
- AI extraction (OCR + LLM)
- ETL pipeline (extract → validate → load)
- Multi-tenant database schemas (demo tenants)
- Streamlit analytics dashboard
- AI assistant: natural-language → SQL

## Phase 2 — Future SaaS Version

- Support contracts, bank statements, ESG reports
- Workflow automation & approvals
- API & SFTP ingestion from external systems
- ESG/CSRD intelligence & gap analysis
- ML-based anomaly detection
- RBAC, admin panel, subscription model
